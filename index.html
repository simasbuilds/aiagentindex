Run of show

The path from idea to prototype.

A guided 90-minute build, followed by room for questions, connections, and next steps.

  1. 00–15

    Frame the opportunity

    What agents are, where they create value, and what makes a good first use case.

  2. 15–35

    Learn the building blocks

    Models, prompts, context, tools, APIs, connectors, and MCP in plain English.

  3. 35–65

    Build the workflow

    Turn a business process into a practical agent loop with boundaries and feedback.

  4. 65–90

    Prototype and refine

    Test the behavior, inspect failure modes, and leave with a repeatable next step.
